STEP 1: Start Backend Server
────────────────────────────────────────────────────────────────────────────
Open Terminal 1 and run:

  cd /home/code/hrms-app/server
  npm start

Expected output:
  ✓ Server running on port 5000
  ✓ Database connected
  ✓ Ready to accept requests

STEP 2: Start Frontend Server
────────────────────────────────────────────────────────────────────────────
Open Terminal 2 and run:

  cd /home/code/hrms-app/client
  npm start

Expected output:
  ✓ Application running on http://localhost:3000
  ✓ Browser opens automatically

STEP 3: Open Application
────────────────────────────────────────────────────────────────────────────
  → Open http://localhost:3000 in your browser
  → Login page should appear

STEP 4: Login and Explore
────────────────────────────────────────────────────────────────────────────
  → Login with admin credentials
  → See Admin Dashboard (Purple theme)
  → Click on different menu items
  → Explore features

STEP 5: Test Different Roles
────────────────────────────────────────────────────────────────────────────
  → Logout
  → Login with HR Manager credentials
  → See HR Manager Dashboard (Dark Purple theme)
  → Repeat for Manager and Employee roles

Problem: Application won't load
Solution:
  1. Check if backend is running: http://localhost:5000
  2. Check if frontend is running: http://localhost:3000
  3. Check browser console (F12) for errors
  4. Restart both servers

Problem: Login fails
Solution:
  1. Verify backend is running
  2. Check credentials are correct
  3. Check browser console for errors
  4. Verify database connection

Problem: Dashboard not showing
Solution:
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Reload page (Ctrl+R)
  3. Logout and login again
  4. Check browser console for errors

Problem: Menu items not showing
Solution:
  1. Verify user role is set correctly
  2. Check JWT token in browser storage
  3. Logout and login again
  4. Check browser console for errors

Problem: Styling looks wrong
Solution:
  1. Clear browser cache
  2. Reload page
  3. Check browser zoom (should be 100%)
  4. Rebuild frontend:
     - Stop frontend server
     - Delete node_modules and package-lock.json
     - Run npm install
     - Run npm start
