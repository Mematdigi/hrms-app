================================================================================
                            API ENDPOINTS
================================================================================

AUTHENTICATION:
   POST   /api/auth/register        - Register new user
   POST   /api/auth/login           - Login user
   GET    /api/auth/profile         - Get user profile

EMPLOYEES:
   GET    /api/employees            - Get all employees
   GET    /api/employees/:id        - Get employee by ID
   POST   /api/employees            - Create employee (Admin/HR)
   PUT    /api/employees/:id        - Update employee (Admin/HR)
   DELETE /api/employees/:id        - Delete employee (Admin)

ATTENDANCE:
   POST   /api/attendance/check-in  - Check in
   POST   /api/attendance/check-out - Check out
   GET    /api/attendance           - Get attendance records
   POST   /api/attendance/mark      - Mark attendance (Admin/HR/Manager)

LEAVE:
   POST   /api/leave/apply          - Apply for leave
   GET    /api/leave                - Get leave requests
   POST   /api/leave/approve        - Approve leave (Admin/HR/Manager)
   POST   /api/leave/reject         - Reject leave (Admin/HR/Manager)

PAYROLL:
   POST   /api/payroll/generate     - Generate payroll (Admin/HR)
   GET    /api/payroll              - Get payroll records
   POST   /api/payroll/process      - Process payroll (Admin/HR)
   POST   /api/payroll/pay          - Mark as paid (Admin/HR)

PERFORMANCE:
   POST   /api/performance/create   - Create review (Admin/HR/Manager)
   GET    /api/performance          - Get reviews
   PUT    /api/performance/:id      - Update review (Admin/HR/Manager)
   POST   /api/performance/submit   - Submit review (Admin/HR/Manager)
