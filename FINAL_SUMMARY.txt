Issue: Application won't load
  Solution: Verify both servers are running
  → Check backend: http://localhost:5000
  → Check frontend: http://localhost:3000

Issue: Login fails
  Solution: Check credentials and backend connection
  → Verify backend is running
  → Check browser console (F12) for errors
  → Verify database connection

Issue: Dashboard not showing
  Solution: Clear browser cache and reload
  → Press Ctrl+Shift+Delete (or Cmd+Shift+Delete on Mac)
  → Clear cache and cookies
  → Reload page

Issue: Menu items not showing
  Solution: Verify user role is set correctly
  → Check JWT token in browser storage
  → Verify role in token matches user role
  → Logout and login again

Issue: Responsive design not working
  Solution: Check browser zoom level
  → Set zoom to 100%
  → Resize browser window
  → Test on actual mobile device

Issue: Styling looks wrong
  Solution: Rebuild frontend
  → Stop frontend server
  → Delete node_modules and package-lock.json
  → Run npm install
  → Run npm start
