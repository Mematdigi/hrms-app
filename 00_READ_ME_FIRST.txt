🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: Application won't load
  → Check both servers are running
  → Check http://localhost:5000 (backend)
  → Check http://localhost:3000 (frontend)

Problem: Login fails
  → Verify backend is running
  → Check credentials
  → Check browser console (F12)

Problem: Dashboard not showing
  → Clear browser cache (Ctrl+Shift+Delete)
  → Reload page (Ctrl+R)
  → Logout and login again

Problem: Menu items not showing
  → Verify user role is correct
  → Check JWT token in storage
  → Logout and login again

Problem: Styling looks wrong
  → Clear cache and reload
  → Check browser zoom (100%)
  → Rebuild frontend if needed
